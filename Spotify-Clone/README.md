# Spotify Clone

A Spotify Web Player Clone built using HTML5 and CSS3. This project recreates the basic user interface of Spotify's web player and was created to practice front-end web development.

---

## Features

* Spotify-inspired user interface
* Sidebar navigation
* Sticky navigation bar
* Music player layout
* Album and playlist cards
* Custom progress bar
* Font Awesome icons
* Clean and modern UI

---

## Technologies Used

* HTML5
* CSS3
* Google Fonts (Montserrat)
* Font Awesome

## How to Run

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser or run it using Live Server.
